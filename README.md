# facebook-clone

### Clone da pagina incial do Facebook com sistema de poste por Firebase
## Versão basica

* Html
* Javascript
* React
* Firebase v8
* CSS

<hr/>

* Comandos para iniciar a aplicação:<br/>

  <p><b>
   npm install react-icons --save<br/>
   npm install firebase@8.10.0<br/>
   npm start<br/>
  <b></p>
